TPL-Typst
------
Collection of templates for typst.


Instructions
----
Clone:
- run cmd `git clone git@github.com:Iardo/tpl-typst.git`, do not use `--recursive`
- move all files inside `tpl-typst`, except the `.git` directory, to your new project root directory
- delete `tpl-typst` directory
- delete `.gitignore` and `readme.txt`
- rename `.gitignore.example` to `.gitignore`
- rename `readme.example.md` to `.readme.md`
- run cmd `git submodule add https://github.com/Iardo/gui-typst .\vendor\gui-typst`
